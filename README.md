# Data Immersion & Wrangling: From Raw Data to Analysis-Ready Dataset

## Overview
This project focuses on acquiring, cleaning, and transforming real-world data to make it **analysis-ready**. The dataset includes customer information, transactions, and demographics, with common issues like missing values, duplicates, and inconsistent formatting.

## Project Objectives
- Understand dataset structure and business relevance
- Create a Data Dictionary
- Perform data profiling and identify quality issues
- Clean and transform the dataset using Python
- Feature engineering (e.g., customer age)
- Deliver a clean, analysis-ready dataset

## Dataset
- `data/raw_data.xlsx` - Original dataset with 100 records
- `data_dictionary.xlsx` - Detailed description of all columns

## Project Workflow
1. **Data Understanding** - Explore the dataset and document all variables.
2. **Data Quality Assessment** - Identify missing values, duplicates, and inconsistencies.
3. **Data Cleaning & Transformation** - Use Python to clean data and generate new features.
4. **Final Output** - Export cleaned dataset as `cleaned_data.xlsx`.

## Scripts
- `scripts/data_cleaning.py` - Python script to clean, transform, and prepare the dataset.

## Tools & Technologies
- Python (Pandas, NumPy, OpenPyXL)
- Excel for data dictionary
- GitHub for version control
- Google Colab or VS Code for execution
